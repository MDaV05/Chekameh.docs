# Features

## ✨ Current Features of Persian Poetry Map

This document outlines the key features implemented in the Persian Poetry Map project.

## 🗺️ Core Features

### Interactive Map Interface

- **MapLibre GL JS Integration** - Smooth, performant map rendering
- **Pan & Zoom Controls** - Intuitive navigation across the geographic area
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **RTL Support** - Full right-to-left language support for Persian text

### Time Slider

- **6 Historical Eras** - Navigate through major periods of Persian poetry
- **150-Year Intervals** - Granular control over time periods
- **Smooth Transitions** - Animated changes between eras
- **Visual Timeline** - Clear indication of current period

### City Markers

- **Interactive Points** - Click to reveal detailed information
- **Hierarchical Styling** - Bold markers for major cultural centers
- **Rich Information Panels** - History, poets, and landmarks
- **Dynamic Visibility** - Cities appear based on selected era

### Poet Information System

- **Biographical Data** - Life stories and historical context
- **Major Works** - Lists of famous poems and books
- **Era Associations** - Connection to specific time periods
- **Geographic Links** - Cities where poets lived and worked

## 🎨 Design Features

### Visual Elements

- **Bootstrap 5 RTL** - Modern, responsive UI framework
- **Vazirmatn Font** - Beautiful Persian typography
- **Lalezar Font** - Decorative headings and titles
- **Color-Coded Information** - Easy visual distinction between content types

### User Experience

- **Intuitive Navigation** - Easy-to-use interface for all ages
- **Fast Loading** - Optimized assets and lazy loading
- **Accessibility** - Designed for inclusive use
- **Cross-Browser Support** - Works on all modern browsers

## 🔧 Technical Features

### Performance

- **Lazy Loading** - Images and content load on demand
- **Asset Optimization** - Minified CSS and JavaScript
- **Efficient Data Handling** - Optimized GeoJSON for maps
- **Caching Strategy** - Browser caching for faster load times
