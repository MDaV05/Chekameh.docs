# مستندات فنی چکامه | Chekameh Technical Docs

به مستندات فنی پروژه **چکامه — نقشه‌ی تعاملی شعر پارسی** خوش آمدید. این سند راهنمای توسعه‌دهندگان، معماران سیستم و مشارکت‌کنندگان فنی است تا با زیرساخت، معماری و نحوه‌ی توسعه‌ی این پلتفرم آشنا شوند.

## 🏗️ معماری کلی سیستم

پروژه چکامه بر پایه‌ی یک معماری ترکیبی (Hybrid) طراحی شده است که بخش فرانت‌اند را به صورت استاتیک و بخش هوش مصنوعی را به صورت Serverless مدیریت می‌کند:

-   **فرانت‌اند:** یک وب‌اپلیکیشن تک‌صفحه‌ای (SPA) مبتنی بر HTML/CSS/JS خالص بدون وابستگی به فریم‌ورک‌های سنگین.
-   **نقشه‌ی تعاملی:** استفاده از کتابخانه‌ی سبک و متن‌باز **MapLibre GL JS** برای رندرینگ برداری نقشه‌ها و مرزهای تاریخی.

## 🛠️ Tech Stack


| Section | Tech |
