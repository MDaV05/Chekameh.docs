# About

## 📖 Project Overview

chekameh is an interactive web application designed to showcase the rich heritage of Persian poetry and its influence across different historical periods and geographical regions.

## 🎯 Mission
 This project is built by team of students passionate about perisna literature and poetry,
 our goal was creating a tool fit for educational purposes.
 making it visually appealing, easy to work with and above all fun although informative, 
 was our end goal which we believe we've done a good work to make it true.
